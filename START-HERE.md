# Nilou Afshar Website

This ZIP contains the source code for the modern Nilou Afshar portfolio website.

## Run locally

Requirements: Node.js 22 or newer.

```bash
npm install
npm run dev
```

Then open the local address shown in the terminal.

## Production build

```bash
npm run build
```

## Important note

The artwork images are currently loaded from `nilouafshar.com`. Before replacing
or shutting down the original website, copy authorized image assets into this
project and update the image URLs in `app/page.tsx`.

The Sites deployment identity file and generated dependency/build caches are not
included in this download package.
